# flutter_background_audio

a simple flutter plugin that supports play audio at background.

I am a fresher in ios and android, so the code maybe a little ugly.

**ios need enable Background Audio** , please reference [Enabling Background Audio](https://developer.apple.com/documentation/avfoundation/media_assets_playback_and_editing/creating_a_basic_video_player_ios_and_tvos/enabling_background_audio)
